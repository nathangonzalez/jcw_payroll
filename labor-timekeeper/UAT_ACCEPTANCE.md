# UAT Acceptance Criteria - Weekly Payroll Policy

## Scope

This UAT pack is for the weekly payroll print/report behavior in admin.

## Acceptance Criteria

- `AC-1` Admin unlock works and the admin dashboard loads.
- `AC-2` Tester can select and load a target payroll week.
- `AC-3` Weekly report flow is visible (`All Entries` panel can be opened for the selected week).
- `AC-4` Weekly payroll print excludes admin roles (Chris Jacobi, Chris Zavesky) with no include-admin toggle.
- `AC-5` Monthly preview includes weekly summary cards (`Week of ...`) so weekly totals are visible before approval.
- `AC-6` Weekly payroll print packet is sorted by pay rate from lowest to highest.

## Out of Scope

- Admin monthly replacement template formatting details.
- Billing report behavior and billing-rate policy changes.
